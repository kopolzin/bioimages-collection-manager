Generate Website

This screen provides advanced features that primarily only the website administrator for Bioimages will have use for.

Resize images

A function is provided to resize the full-size images down to the various resolutions used on the Bioimages website. When resized to thumbnails (tn) the larger image dimension is scaled down to 100 pixels, low quality (lq) images are scaled down to 480 pixels, and good quality (gq) images down to 1024 pixels.

The namespace is the name of the folder that the images will reside in on the Bioimages server. Typically it is the agent ID of the photographer or their organization's ID. When images are resized they will be saved to the following folders: <base_output_directory>/tn/<namespace>/, <base_output_directory>/lq/<namespace>/, and <base_output_directory>/gq/<namespace>/.
